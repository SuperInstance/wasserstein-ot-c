# PLUG_AND_PLAY — wasserstein-ot-c

> *Integration guide.*


## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library |

## Integration

Import `wasserstein_ot_c` in your project to access the functionality.

```

```
